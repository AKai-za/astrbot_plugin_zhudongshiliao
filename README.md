# 自动私聊插件 (astrbot_plugin_zhudongshiliao)

AstrBot 自动私聊插件，支持自动私聊用户、群消息总结、错误信息转发等功能。

## 功能特性

- **自动私聊**：当用户发送包含特定关键词的消息时，自动触发私聊
- **群消息总结**：自动记录群消息历史，并可根据关键词或指令触发总结
- **错误信息转发**：将插件执行过程中的错误信息自动转发给指定用户
- **主动私聊**：通过指令主动触发对特定用户的私聊
- **WebUI 配置**：支持在管理面板上直接配置插件参数

## 指令系统

- `/private <用户ID> <消息内容>`：主动触发对指定用户的私聊
- `/summary [群ID]`：手动触发群消息总结，可指定群ID
- `/set_target <用户ID>`：设置错误信息转发目标
- `/config`：查看当前插件配置

## 配置项

通过 WebUI 管理面板或配置文件可设置以下参数：

- **错误信息转发目标**：接收错误信息的用户ID
- **群消息总结触发关键词**：触发群消息总结的关键词列表
- **私聊触发关键词**：触发自动私聊的关键词列表
- **是否自动总结群消息**：是否开启自动总结功能
- **自动总结间隔**：自动总结群消息的时间间隔（秒）

## 安装方法

1. 将插件目录 `astrbot_plugin_zhudongshiliao` 复制到 AstrBot 的 `plugins` 目录
2. 启动 AstrBot，插件会自动加载
3. 在管理面板上配置插件参数

## 使用示例

### 1. 自动私聊
在群里发送包含关键词的消息，如：
```
你好
```
机器人会自动私聊你。

### 2. 群消息总结
在群里发送包含关键词的消息，如：
```
总结
```
机器人会将群消息总结发送到你的私聊。

### 3. 主动私聊
使用指令触发主动私聊，如：
```
/private 123456 你好，这是一条测试消息
```
机器人会向用户 123456 发送私聊消息。

### 4. 设置错误信息转发目标
使用指令设置错误信息转发目标，如：
```
/set_target 123456
```
之后插件的错误信息会自动转发给用户 123456。
